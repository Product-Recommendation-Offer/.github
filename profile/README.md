Welcome to the GitHub page of **AC-01 Telco!**

## Background

Banyak perusahaan telekomunikasi menawarkan berbagai promo dan paket layanan kepada pelanggan, namun sering kali promosi tersebut tidak sesuai dengan kebutuhan dan perilaku pengguna. Akibatnya, tingkat respons pelanggan terhadap promo rendah, peluang cross-selling dan up-selling tidak optimal, serta risiko pelanggan berpindah ke kompetitor meningkat.

Perusahaan Telco memiliki banyak data berharga seperti profil pelanggan (umur, lokasi, paket aktif, lama berlangganan), perilaku penggunaan (internet, telepon, SMS, pengeluaran), serta interaksi terhadap promo sebelumnya. Sayangnya, data tersebut sering belum dimanfaatkan secara optimal untuk mendukung keputusan promosi yang berbasis data. Melalui penerapan machine learning, data tersebut dapat diolah untuk menghasilkan rekomendasi produk yang relevan secara otomatis.
